# SIMONETv2
SIMONET (Sistem Monitoring Network) adalah sebuah aplikasi monitoring jaringan yang berbasis PHP dengan framework CodeIgniter dengan menggunakan MikroTik API dan UniFi API sebagai penghubung antara Perangkat Jaringan dengan Aplikasi.
